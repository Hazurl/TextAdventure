# Items
This type of file contains json objects which describe items

## Example
```json
    "revolver" : {
        "name" : "Colt Walker Revolver",
        "description" : "Very big and bulky, however powerful revolver"
    },

    "gold_key" : {
        "name" : "Gold key",
        "description" : "Key given by an old miner"
    }
```
